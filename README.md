## 🧑‍💻 Run Locally

> Requires [Node.js](https://nodejs.org) 18+ and npm.

```bash
# 1. Install dependencies
npm install

# 2. Start the dev server (http://localhost:4200)
npm start

# 3. Production build → dist/portfolio/browser
npm run build
```

## 🌐 Deployment

The production build outputs a static site to `dist/portfolio/browser`, ready to deploy on any static host (GitHub Pages, Netlify, Vercel, Firebase Hosting, AWS S3 + CloudFront).

```bash
npm run build
# then publish the contents of dist/portfolio/browser
```

> **SPA note:** configure your host to redirect all routes to `index.html` so client-side routing works on refresh/deep links.
